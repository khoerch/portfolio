Hi there! My name is Kevin and I am a fullstack developer solving problems in Vermont.

BIO

I am a web developer and recent graduate of the Web Developer Track at Bloc.io. Board games inspired my interest in programming. My favorite part of any game is reading and explaining the rules. I approach any web development problem like a complex game -- breaking down each requirement and writing rules to govern that logic.

After four years in Washington, D.C., my partner and I moved to Vermont to be closer to the outdoors. When I’m not working I’m often snowboarding, climbing, camping, or reading comics. We make a goal of finding new places to explore each season. I take the same approach to web development. I am always eager to learn something new and make time to explore beyond my comfort zone.

PROJECTS

Title: Spider-Man Quiz App

Project description: This is an early project to combine my experience in HTML, CSS, and JavaScript. The goal was to create a simple web app on everyone’s friendly neighborhood Spider-Man. I focused on writing clean, descriptive JavaScript that over users could easily read and understand. The HTML is designed to be as accessible as possible, following a11y best practices. I also hope it inspires some quiz-takers to pick up a comic!

Link to live app: https://khoerch.github.io/quiz-app/

Link to repo: https://github.com/khoerch/quiz-app

Technologies Used: HTML, CSS, JavaScript, jQuery
